# Agent: Constraints

## Scope

- Do NOT expand scope beyond the acceptance criteria
- Do NOT ask questions — the task is already refined to 100% confidence
- Do NOT modify files outside the scope described in the task
- If the task says "that's it" — stop. Do not add extra steps.

## Blockers

- If you hit an unresolvable blocker, exit with a clear error message describing the issue
- Do NOT retry the same failing approach more than twice
- Do NOT investigate unrelated systems to work around a blocker

```

```
