# Expected automation artifacts

The envctl database automation must preserve the original migration artifacts and add automation artifacts:

- Target descriptor.
- Artifact contract version.
- Migration recipe version.
- Run ledger.
- Event timeline.
- Operation queue.
- Approval ledger.
- Evidence register.
- Artifact registry.
- Link graph.
- Validation scorecard.
- Replay readiness report.
- Rollback readiness report.
- Human involvement transcript.
- Agent session ledger.
- Plugin session ledger.
